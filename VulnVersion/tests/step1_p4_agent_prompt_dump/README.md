# Step1 P4 Agent Prompt Dump

This directory contains exact system and prompt text files sent to OpenCode during the 18-CVE P4 A/B runs.

- `step1_p4_ab_packet_only_18cve/`: packet-only prompts and system prompts.
- `step1_p4_ab_git_tools_18cve/`: packet + read-only git tools prompts and system prompts.
- `index.json`: file list with byte sizes.

The files are copied verbatim from each CVE `step1/agent_calls/` directory.
