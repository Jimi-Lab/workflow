# CWE-792 Family View
Name: Incomplete Filtering of One or More Instances of Special Elements

## Family Summary

The product receives data from an upstream component, but does not completely filter one or more instances of special elements before sending it to a downstream component.

## Use This Family View When

- only a broad CWE family is known
- you need to choose a more specific child CWE before loading a stage file
- the available CVE metadata is vague or only maps to a parent class

- `CWE-793`: Only Filtering One Instance of a Special Element (Variant)
- `CWE-794`: Incomplete Filtering of Multiple Instances of Special Elements (Variant)
