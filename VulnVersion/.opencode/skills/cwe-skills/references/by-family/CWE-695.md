# CWE-695 Family View
Name: Use of Low-Level Functionality

## Family Summary

The product uses low-level functionality that is explicitly prohibited by the framework or specification under which the product is supposed to operate.

## Use This Family View When

- only a broad CWE family is known
- you need to choose a more specific child CWE before loading a stage file
- the available CVE metadata is vague or only maps to a parent class
