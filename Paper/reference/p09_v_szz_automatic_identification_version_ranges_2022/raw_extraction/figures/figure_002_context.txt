some CVE vulnerabilities is spurious, and the vulnerable versions
can be refined based on the fixing commits and the inducing com-
mits generated by SZZ algorithms. However, the second example
reveals that SZZ algorithms might not identify the true inducing
commit. Because SZZ algorithms assume the commit that last mod-
