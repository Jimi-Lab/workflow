4.5 RQ3. Resource Consumption
We implement CaVulner by invoking LLM APIs. Since analyzing the root cause of a vulnerability
and localizing the corresponding code blocks requires a non-trivial amount of time and tokens, we
assess the cost of LLM usage. As shown in Figure 2,CaVulner relies on LLM in three stages: Coarse-
grained Vulnerable Methods Localization, Fine-grained Vulnerable Code Block Identification, and
Vulnerability-inducing Commit Tracing. We evaluate the token consumption, and time overhead
during these stages.
