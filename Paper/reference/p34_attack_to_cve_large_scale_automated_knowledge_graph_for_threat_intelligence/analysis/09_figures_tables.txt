Useful figure/table patterns:
- Pipeline/schema figure: ingestion from CVE/CWE/CAPEC/ATT&CK into a unified KG.
- Path visualization: CVE -> CWE -> CAPEC -> ATT&CK and reverse ATT&CK -> CVE paths.
- Technique/group neighborhood visualization: a graph-centered view around one technique or intrusion set.
- Entity/relationship count tables: compactly summarize graph scale and coverage.
- Case-study tables: list technique, CAPEC IDs, CWE IDs, and recovered CVEs for selected groups or advisories.
