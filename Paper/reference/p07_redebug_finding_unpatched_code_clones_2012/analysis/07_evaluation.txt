Main Results:
The paper reports that ReDeBug scales to billions of lines, finds thousands of unpatched code clones, and confirms real bugs. Local reproduction was not requested.

Per-RQ Findings:
Attacker/patch-release question:
- Released patches can expose many cloned vulnerable regions across large source corpora.
OS responsiveness question:
- Later distribution versions still contain unpatched code clones for older and newer security patches.
Persistence question:
- Unpatched clones can persist across distribution versions and across many years of patches.
