4 Implementation of M OVERY
In this section, we introduce the implementation of MOVERY ,
including the vulnerability dataset and the architecture.
4.1 Vulnerability dataset
Collecting security patches. We collected security patches
by leveraging the method used in the previous approaches [19,
32]; we examined CVEs in the National Vulnerability
Database (NVD) and checked ifgit commitURLs (i.e., secu-
rity patch commits) were included in the references. We then
gathered security patches by crawling such patch commits
from the corresponding Git repositories. We chose the C/C++
vulnerabilities as our initial targets, because code fragments
reuse and modified OSS reuse are prominent in C/C++ soft-
ware [17, 47–49]. Consequently, we collected 4,219 C/C++
security patches from the NVD (as of March 2021).
